**caldav.go** — CalDAV sync engine:
- `CalDAVSync` struct: Source (*caldav.Client), Dest (*caldav.Client), State (*MigrationState), StatePath (string), OnProgress (func(calendar string, current, total int)), OnCalendarStart (func(calendar string, total int)), OnCalendarDone (func(calendar string))
- `NewCalDAVSync(source, dest *caldav.Client, state *MigrationState, statePath string) *CalDAVSync`
- `DiscoverCalendars(ctx context.Context, principal string) ([]CalendarInfo, error)` — Find all calendars on source. CalendarInfo: Name (string), Path (string), EventCount (int)
- `SyncCalendar(ctx context.Context, sourcePath, destPath string) (*CalSyncResult, error)` — Sync one calendar:
  1. Query all events from source calendar using CalendarQuery with AllProp
  2. For each event: extract UID from iCal data
  3. Check state.IsCalEventMigrated(uid); skip if done
  4. PUT event to destination calendar path
  5. Mark migrated, call OnProgress, save state periodically
  6. On error: log and continue (skip-and-report)
  7. Return CalSyncResult: Calendar (string), Migrated (int), Skipped (int), Errors (int)
- `SyncAll(ctx context.Context, sourcePrincipal, destPrincipal string) (*CalSyncAllResult, error)` — Discover and sync all calendars
- `DryRun(ctx context.Context, sourcePrincipal string) (*CalDryRunResult, error)` — List calendars and event counts without syncing

**carddav.go** — CardDAV sync with contact merge:
- `CardDAVSync` struct: Source (*carddav.Client), Dest (*carddav.Client), State (*MigrationState), StatePath (string), MergeMode (string: "append", "overwrite", "skip"), OnProgress (func(book string, current, total int))
- `NewCardDAVSync(source, dest *carddav.Client, state *MigrationState, statePath string) *CardDAVSync`
- `DiscoverAddressBooks(ctx context.Context, principal string) ([]AddressBookInfo, error)` — Find all address books. AddressBookInfo: Name (string), Path (string), ContactCount (int)
- `SyncAddressBook(ctx context.Context, sourcePath, destPath string) (*CardSyncResult, error)` — Sync one address book:
  1. Query all contacts from source using AddressBookQuery with AllProp
  2. Query all contacts from destination (for merge detection)
  3. For each source contact: extract primary email from vCard
  4. Check state.IsContactMigrated(email); skip if done
  5. Check if contact exists in destination (match by email per locked decision)
  6. If exists and MergeMode=="append": call mergeContact() — fill empty fields from source, don't overwrite per locked decision
  7. If exists and MergeMode=="overwrite": replace destination with source
  8. If exists and MergeMode=="skip": skip entirely
  9. If new: PUT contact to destination
  10. Mark migrated, save state periodically
  11. Return CardSyncResult: Book (string), Created (int), Merged (int), Skipped (int), Errors (int)
- `mergeContact(dest, source *vcard.Card) (*vcard.Card, []string)` — Merge logic per locked decision:
  - For each field in source (TEL, ADR, BDAY, NOTE, ORG, TITLE, etc.)
  - If destination field is empty → copy from source
  - If destination field has a value → keep destination (don't overwrite)
  - Return merged card and list of merge decisions for logging
- `SyncAll(ctx context.Context, sourcePrincipal, destPrincipal string) (*CardSyncAllResult, error)`
- `DryRun(ctx context.Context, sourcePrincipal string) (*CardDryRunResult, error)` — List address books and contact counts

Tests:

- CardDAV: test mergeContact logic thoroughly — empty field fills from source, existing field preserved, multiple emails, all merge modes
- State integration: verify IsCalEventMigrated/MarkCalEventMigrated called correctly

**fileimport.go** — Import contacts from .vcf files and calendars from .ics files:

VCF Import:
- `ImportVCF(ctx context.Context, filePath string, dest *carddav.Client, destPath string, state *MigrationState, statePath string, mergeMode string, onProgress func(current, total int)) (*VCFImportResult, error)`
  1. Read .vcf file (may contain multiple vCards separated by BEGIN:VCARD/END:VCARD)
  2. Parse using emersion/go-vcard Decoder
  3. For each vCard: extract primary email
  4. Check state, apply merge logic (same as CardDAVSync.mergeContact), PUT to destination
  5. Save state periodically
  6. Return VCFImportResult: Total (int), Created (int), Merged (int), Skipped (int), Errors (int)
- `DryRunVCF(filePath string) (*VCFDryRunResult, error)` — Parse file, count contacts, show preview without importing
  - VCFDryRunResult: Total (int), WithEmail (int), WithoutEmail (int) — contacts without email can't be matched for merge

ICS Import:
- `ImportICS(ctx context.Context, filePath string, dest *caldav.Client, destPath string, state *MigrationState, statePath string, onProgress func(current, total int)) (*ICSImportResult, error)`
  1. Read .ics file (may contain multiple VEVENT/VTODO blocks in one VCALENDAR)
  2. Parse using emersion/go-ical Decoder
  3. For each event: extract UID
  4. Check state.IsCalEventMigrated(uid); skip if done
  5. Wrap individual event in VCALENDAR envelope for PUT
  6. PUT to destination CalDAV server
  7. Save state periodically
  8. Return ICSImportResult: Total (int), Imported (int), Skipped (int), Errors (int)
- `DryRunICS(filePath string) (*ICSDryRunResult, error)` — Parse file, count events/todos, show preview

Helper functions:
- `extractPrimaryEmail(card *vcard.Card) string` — Get first EMAIL field value from vCard
- `parseVCFFile(path string) ([]*vcard.Card, error)` — Read and parse multi-vCard file
- `parseICSFile(path string) ([]*ical.Calendar, error)` — Read and parse ICS file

Tests:
- VCF import: test with multi-contact .vcf content (create test fixtures as string constants), verify parse count, verify merge logic applied, verify state tracking
- ICS import: test with multi-event .ics content, verify parse count, verify state tracking, verify VCALENDAR envelope wrapping
- DryRun: verify counts without side effects
- Edge cases: empty file, single contact, contact without email
